## Tips for Category Usage

1. **Use primary category for focused results:**
   ```python
   query = "cat:cs.LG"  # Primary and cross-listed
   ```

2. **Combine categories for interdisciplinary topics:**
   ```python
   query = "cat:cs.LG OR cat:q-bio.QM"  # ML + computational biology
   ```

3. **Use wildcards (when available):**
   Some queries support wildcards for category groups.

4. **Check paper categories:**
   Papers are often cross-listed in multiple categories. Check `categories` field in results.

5. **Category vs. keyword search:**
   Category filtering is faster and more precise than keyword-only searches.
